##usage

1. ./run_alert_app_conda.sh

(Script automatically handles conda env creation, activation, and dependency installation)


##rerun after changes / troubleshooting

If port 8501 is already in use:
1. netstat -tlnp | grep 8501
2. kill -9 <PID> # kill the process using the port
3. ./run_alert_app_conda.sh

Or use FAST mode to skip dependency checks:
FAST=1 ./run_alert_app_conda.sh
